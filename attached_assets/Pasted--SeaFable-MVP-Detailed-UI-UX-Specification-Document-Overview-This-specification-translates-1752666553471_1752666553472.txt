**Visual Requirements:**
- **Location**: Centered in hero section
- **Size**: Full-width on mobile, max-width 800px on desktop
- **Style**: Clean white background, rounded corners
- **Fields**: 3 main fields + search button
- **Responsive**: Stack vertically on mobile

**Functionality (P0):**
- **What Field**:

  - **P1 Enhancement**: Add suggested categories dropdown
- **Where Field**: 
  - Free text input for location
  - **P1 Enhancement**: Add suggested places based on operation areas
- **Date Field**: 
  - Simple date picker
  - **P1 Enhancement**: Add duration selector (hours/days)
- **Search Button**: Navigate to search results page

## üìã 8. Technical Implementation Notes

### 8.1 Performance Requirements

#### **Loading Times**
- Homepage: < 3 seconds
- Search results: < 2 seconds
- Experience details: < 2 seconds
- Booking flow: < 1 second per step

#### **Optimization**
- Image lazy loading
- Code splitting by route
- API response caching
- Progressive image loading

### 8.2 SEO Requirements

#### **Meta Tags**
```tsx
<Head>
  <title>SeaFable - Book Water Adventures Worldwide</title>
  <meta name="description" content="Discover and book amazing water adventures..." />
  <meta property="og:title" content="SeaFable" />
  <meta property="og:description" content="..." />
  <meta property="og:image" content="/og-image.jpg" />
</Head>
```

### 8.3 Accessibility Requirements

#### **WCAG 2.1 AA Compliance**
- Keyboard navigation support
- Screen reader compatibility
- Color contrast ratios > 4.5:1
- Alt text for all images
- Semantic HTML structure
